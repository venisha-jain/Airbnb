Global Airbnb Performance Dashboard – Power BI Project
📌 Project Overview

The Global Airbnb Performance Dashboard is an interactive Power BI project designed to analyze Airbnb listings, host behavior, customer reviews, pricing trends, and trust metrics across major global cities.

This dashboard provides business insights into:
Market share by city
Listing growth trends
Property type performance
Customer review behavior
Host trust verification
Pricing analysis
Rating comparison between cities

The project helps understand Airbnb’s expansion, customer engagement, and host reliability across international markets.
